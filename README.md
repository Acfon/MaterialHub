<h1 align="center">MaterialHub</h1>
<h3 align="left">MaterialHub - проект представляет собой приложение для автоматизации учета и управления запасами пластика для 3D-принтеров на складе или в мастерской. Система предназначена для устранения хаоса в учете материалов, предотвращения простоев печати из-за нехватки филамента и оптимизации процессов закупок. Данное приложение позволит перевести учет пластика на качественно новый уровень, сэкономив время, деньги и нервы сотрудников, и сосредоточиться на самом главном — на эффективной 3D-печати.</h3>
<h1 align="left">Цель:</h1>
<h3 align="left">Создать приложение для ведения учёта расходных материалов на складе для 3D
принтеров, с возможностью добавления и вычета материалов на складе по разным
характеристикам и свойствам пластика.</h3>
<h1 align="left">Задачи:</h1>
<h3 align="left">1. Разработка структуры</h3>
<h3 align="left">2. Создание дизайна</h3>
<h3 align="left">3. Создание приложения на языке Python</h3>
<h3 align="left">4. Разработка системы фильтрации</h3>
<h3 align="left">5. Исправление багов</h3>
<h1 align="left">Функционал:</h1>
<h3 align="left">Основной функцией является добавление и экспортирование файлов Excel, а также функция фильтрации таблицы. Также в приложении легко можно изменить и добавить новые данные
</h3>
<h3 align="left">Краткие тезисы:
  <h3 align="left">1. Модуль учета материалов</h3>
  <h3 align="left">2. Фильтрация и поиск</h3>
  <h3 align="left">3. Визуализация и отчетность</h3>
  <h3 align="left">4. Исправление проблем из вашей таблицы</h3>
<h3 align="left">Импорт и экспорт легко осуществляется при помощи нажатия кнопок. Для того, чтобы отредактировать материал, необходимо выбрать его в таблице, а потом нажать на кнопку "Изменить". Также присутсвует функция обновления отображаемых данных в таблице. Если нажать на значение шапки таблицы, то появится список индивидуальных значений, выбор которых которых будет фильтровать таблицу.
